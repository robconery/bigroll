## The Commands

To find out Ruby version:

```sh
ruby -v
```

Working with rbenv:

```sh
rbenv versions
rbenv install 3.2.2
rbenv local 3.2.2
```

Creating a Rails app with Postgres and Tailwind:

```sh
rails new railz -d postgresql -c tailwind
cd railz
rails s
bin/dev
```
